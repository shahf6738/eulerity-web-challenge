# Eulerity Web Challenge

A modern React + TypeScript pet gallery application built for the Eulerity Frontend Challenge.

## Features

- Fetch pets from API
- Responsive pet gallery
- Search pets by title or description
- Ascending / descending sorting
- Multi-image selection
- Download selected images
- Estimated download size
- Infinite scrolling
- Pet detail pages with React Router
- Persistent selection state using Context API
- Loading and empty states
- Modern responsive UI
- Styled with styled-components

---

## Tech Stack

- React
- TypeScript
- Vite
- styled-components
- React Router DOM
- Context API
